📋 프로젝트 개요

코딩 에이전트(Claude Code, Codex, Cursor 등)가 UI 화면·기능·인프라·API·코드 심볼·이슈를 UUID 기반으로 안정적으로 참조할 수 있게 하는 경량 저장소. 문서 파싱은 외부 에이전트가 담당하고, 이 서비스는 정리된 결과를 저장·조회·제공하는 역할만 한다.

외부 Parser / Agent │ ▼ POST /ingest/batch ┌─────────────────────────────┐ │ Registry (PostgreSQL 16) │ │ entity · alias · context │ │ relation · tag · revision │ └─────────────────────────────┘ │ │ ▼ REST read/write ▼ MCP read-only (stdio) 관리자 콘솔 Coding Agent (Next.js BFF) (Claude Code 등)

⚠️ 핵심 불변 규칙

Entity UUID 불변
entity.id는 생성 후 절대 변경 금지. PATCH에서 id·type 필드 수신 시 무시.
Type 변경 금지
Ingest 시 기존 entity의 type 변경 시도 → 400 TYPE_CHANGE_FORBIDDEN
Deprecated 삭제 금지
deprecated entity는 DELETE 금지. status=deprecated + replacement_entity_id 기록.
MCP Write 금지
MCP server는 read-only. write tool 추가 금지.
